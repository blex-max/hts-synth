class SamFlag(*values)[source]

Bases: IntFlag

Properties of the SAM flag.

Create a new SAM flag by combining these using bitwise OR, eg:

flag = SamFlag.FIRST_IN_PAIR | SamFlag.READ_MAPPED_IN_PROPER_PAIR | SamFlag.READ_PAIRED
